# DELTARUNE — Packs de traducción

Packs de traducción multi-idioma para **DELTARUNE**, distribuidos para el parcheador
**DeltaruneTranslator**. Los archivos `.langpack` / `.fontpack` / `.imgpack` están
**cifrados** y no son utilizables fuera del parcheador.

- **Cómo usar:** descarga el parcheador → elige idioma y capítulos → parchea.
- **Descargas / soporte:** [Discord](https://discord.new/rnYqEw7Yvr)
